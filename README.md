# Fromvill Website

A static Vite React website for Fromvill with routed sub pages.

## Local commands

```bash
npm install
npm run dev
npm run build
```

The production build outputs to `dist`, ready for Cloudflare Pages. The `public/_redirects` file keeps direct page routes such as `/services` and `/contact` working after deployment.

## Logo

The provided Fromvill logo is included as `public/fromvill-logo.png`. A cropped wordmark from the same logo is used for compact header and footer placement.
